# Baremetal Model

This example demonstrates how to query baremetal model information using the Arsys Baremetal provider.

## What This Example Does

- Queries all available baremetal models using `arsys-baremetal_baremetal_models`
- Gets detailed information about a specific baremetal model using `arsys-baremetal_baremetal_model.example`
- Outputs the list of baremetal models and details of a specific one
