# Public Network

This example demonstrates how to query public network information using the Arsys Baremetal provider.

## What This Example Does

- Queries all available public networks using `arsys-baremetal_public_networks`
- Gets detailed information about a specific public network using `arsys-baremetal_public_network`
- Queries all IPs in a public network using `arsys-baremetal_public_network_ips`
- Gets detailed information about a specific IP in a public network using `arsys-baremetal_public_network_ip`
- Outputs the list of public networks and details of a specific one
- Outputs the list of IPs in a public network and details of a specific one
