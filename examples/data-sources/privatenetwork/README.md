# Private Network

This example demonstrates how to query private network information using the Arsys Baremetal provider.

## What This Example Does

- Queries all available private networks using `arsys-baremetal_private_networks`
- Gets detailed information about a specific private network using `arsys-baremetal_private_network`
- Queries all available servers assigned to a private network using `arsys-baremetal_private_network_servers`
- Gets detailed information about a specific server assigned to a private network using `arsys-baremetal_private_network_server`
- Outputs the list of private networks and details of a specific one
- Outputs the list of servers assigned to a private network and details of a specific one
