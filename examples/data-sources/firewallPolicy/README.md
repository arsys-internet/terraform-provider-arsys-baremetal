# Firewall Policy

This example demonstrates how to query firewall policy information using the Arsys Baremetal provider.

## What This Example Does

- Queries all available firewall policies using `arsys-baremetal_firewall_policies`
- Gets detailed information about a specific firewall policy using `arsys-baremetal_firewall_policy`
- Queries all available firewall policy rules using `arsys-baremetal_firewall_policy_rules`
- Gets detailed information about a specific firewall policy rule using `arsys-baremetal_firewall_policy_rule`
- Outputs the list of firewall policies and details of a specific one
- Outputs the list of firewall policy rules and details of a specific one
